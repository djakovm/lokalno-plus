# Локално+

Локално+ е frontend демо апликација за дигитален каталог и маркетплејс на локални бизниси во Северна Македонија.

## Технологии

- React + TypeScript
- Vite
- React Router
- Tailwind CSS
- Lucide React
- JSON демо податоци и localStorage

## Стартување

```bash
npm install
npm run dev
```

## Продукциски build

```bash
npm run build
npm run preview
```

## Демо сметки

- Купувач: `customer@lokalno.mk` / `demo123`
- Бизнис: `business@lokalno.mk` / `demo123`
- Админ: `admin@lokalno.mk` / `demo123`

## Vercel

Проектот е подготвен за Vercel. `vercel.json` ги пренасочува сите React Router патеки кон `index.html`.
